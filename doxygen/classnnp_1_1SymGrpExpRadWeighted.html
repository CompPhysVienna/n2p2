Detailed Description

Weighted radial symmetry function group (type 12)

\[ G^{12}_i = \sum_{j \neq i} Z_j \mathrm{e}^{-\eta(r_{ij} - r_\mathrm{s})^2} f_c(r_{ij}) \]

Common features:

  • element of central atom
  • cutoff type
  • \(r_c\)
  • \(\alpha\)
